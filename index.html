Ejercicios de JavaScript

1. Aplicar a Crédito Bancario

Ingrese su edad y salario mensual para verificar si puede aplicar a un crédito bancario.

2. Verificar Divisibilidad

Ingrese un número para verificar si es divisible entre 3, 5, o ambos. También se mostrará una lista de todos los números divisibles entre 3 o 5 hasta el número ingresado.

3. Calcular Factorial

Ingrese un número para calcular su factorial.
